# Место
Переписан на React.js

## Функционал:
- Регистрация и авторизация пользователей.
- Добавление и удаление карточек
- Динамическая валидация вводимых данных.
- Измененеие данных пользователя.
- Сохранение собственной  карточек. 

### Стек
* html
* css
* bem
* React.js
* Express.js
* Mongo.db

### Запуск проект
* npm init
* npm start
